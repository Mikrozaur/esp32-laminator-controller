# Контроллер рулонного ламинатора на ESP32-C3

Полностью открытый проект современного контроллера рулонного ламинатора на базе **ESP32-C3** и **ESPHome**.

Проект разработан как замена штатной электронике недорогих рулонных ламинаторов и ориентирован на надежность, ремонтопригодность и использование стандартных компонентов.

---

## Почему появился этот проект

Поводом для разработки стала неисправность штатной платы управления ламинатора.

Вместо ремонта оригинальной электроники было принято решение разработать полностью новый контроллер с современной архитектурой, поддержкой ESPHome и возможностью интеграции с Home Assistant.

В процессе разработки проект неоднократно перерабатывался и в итоге превратился не в копию штатной платы, а в самостоятельный контроллер с собственной схемотехникой и логикой работы.

---

## Основные возможности

- Управление нагревателем
- Управление реверсом двигателя
- Аппаратный MASTER с полным отключением питания исполнительной части
- Стрелочный индикатор температуры
- Поддержка адресной индикации WS2812 (опционально)
- Измерение температуры с помощью стеклянного NTC 100 кОм
- Настройка температуры потенциометром
- Встроенный Web Server
- Интеграция с Home Assistant
- Прошивка на ESPHome
- Полностью модульная конструкция

---

## Особенности проекта

Основной задачей являлось создание максимально простой конструкции, которую можно повторить без изготовления сложной силовой электроники.

Проект построен из стандартных модулей:

- ESP32-C3 LuatOS Core
- стандартный четырехканальный релейный модуль
- отдельный блок питания 220В → 5В
- собственная плата контроллера низкого напряжения

Любой модуль может быть заменен независимо от остальных.

---

## Безопасность

Одной из основных целей проекта было максимально отделить высоковольтную часть устройства от схем управления.

В результате вся логика контроллера работает только с безопасным напряжением **3.3 В** и **5 В**.

Высоковольтная часть (220 В) полностью сосредоточена в отдельных узлах:

- блок питания 220 В → 5 В;
- четырехканальный релейный модуль;
- силовая коммутация нагревателя и двигателя.

На плате контроллера отсутствуют высоковольтные цепи.

Органы управления пользователя (переключатели, потенциометр, индикаторы) также работают только с низким напряжением.

Такое разделение значительно повышает безопасность эксплуатации, упрощает изготовление платы и облегчает ремонт устройства.

| GPIO3 | SEAL |

| GPIO5 | Реле реверса |
| GPIO6 | DIR |

| GPIO20 | LED PREHEAT |

## Аппаратная часть

Контроллер построен на базе ESP32-C3.

Используемые основные компоненты:

- ESP32-C3 LuatOS Core
- AMS1117-3.3
- Стеклянный NTC 100 кОм B4300
- Потенциометр B10K
- Стандартный релейный модуль Active LOW

Все исполнительные устройства управляются через релейный модуль.

---

## Программное обеспечение

Контроллер работает под управлением **ESPHome**.

Реализовано:

- управление нагревателем;
- управление реверсом двигателя;
- измерение температуры;
- настройка температуры;
- управление стрелочным прибором;
- поддержка Home Assistant;
- встроенный Web Server.
